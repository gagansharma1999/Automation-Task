# Automation-Task
Automation code is executed using cypress with cucmber.

Prerequisites
Node.js installed on your machine.
Cypress installed (npm install cypress).

Folder Structure:

- cypress/
   - integration/
      - registration.spec.js
      - login.spec.js
   - support/
      - commands.js
   - plugins/
      - index.js
- cypress.json
- package.json
